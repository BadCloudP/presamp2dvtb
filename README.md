局限：

1.只能根据交叉设定来大概分辨清浊辅音，导出后需要手动处理

2.目前只能读取通用设定，目前并不能支持presamp的特殊设定

3.只能读取字母UTF-8，不支持日语

4.特殊音素也需要手动处理，例如br
